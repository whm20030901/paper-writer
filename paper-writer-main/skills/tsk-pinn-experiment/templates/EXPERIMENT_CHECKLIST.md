# Experiment Checklist

## Before coding
- [ ] Problem statement is explicit
- [ ] PDE benchmark is selected
- [ ] Baselines are selected
- [ ] Success metrics are defined
- [ ] First-pass scope is minimal and runnable

## Before running
- [ ] Config file exists
- [ ] Random seed is fixed
- [ ] Output directory is configured
- [ ] Baseline entrypoint exists
- [ ] Proposed model entrypoint exists
- [ ] Metrics saving is implemented
- [ ] Plot saving is implemented

## After running
- [ ] Final metrics table saved
- [ ] Training curves saved
- [ ] Partition visualization saved
- [ ] Local weight visualization saved
- [ ] Checkpoint saved
- [ ] Markdown report written
- [ ] Underperformance explicitly documented
