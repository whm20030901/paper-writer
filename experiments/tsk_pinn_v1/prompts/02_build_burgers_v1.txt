Use $tsk-pinn-experiment.

Build the first runnable benchmark for the 1D Burgers equation.

Requirements:
1. Implement a baseline vanilla PINN.
2. Implement a TSK-based local reliability weighted PINN.
3. Use a modified fully-connected backbone if practical.
4. Use Gaussian antecedents and a first-order TSK consequent.
5. Save:
   - configs
   - checkpoints
   - training curves
   - metrics JSON/CSV
   - partition visualizations
   - local loss weight visualizations
   - a markdown report
6. If the proposed model underperforms, document it explicitly in a section titled `Not Meeting Expectations`.
